Bottle It Up!

Bottle It Up! takes a user's information from their Facebook profile and allows them to send their own 'message in a bottle' as well as receive one, with all messages being stored and executed out of a particular database. The website was built using Javascript, Firebase, CSS, Bootstrap, and HTML.

ICEE Locator

The ICEE Locator iOS App provides users the ability to find the nearest location carrying ICEE's based on their current location. It was built using Swift and Xcode. The Google Places API was implemented to create a map displaying nearby locations of establishments carrying ICEE's.

Hackers vs. Harassment

Hackers vs. Harassment addresses the continual issue of bullying and online harassment seen within today's society and urges others to take a stand against it. The website was built using Bootstrap, HTML, Javascript, and CSS, and the website's survey was programmed using Python, Flask, and JSON.

Simulating Game of Craps in Python
CS110 Final Project - Emily Lakic, Mark Caldropoli, Jordana Simon, Samantha Chu and Danielle O'Neal

As part of a final project for my CS110 class, I worked in a group of five people to program a game of craps in Python. We used digital wireframing as our public interface. Our classes concerning logic were CrapsGame and Die, and our GUI classes were Craps, CrapsTable, and ControlPanel. My individual contribution to the project was coding the class RollingDie, which delt with both logic and GUI. I helped others debug their individual code, and also received help when coming across a bug in my own. We performed integration testing using various test inputs to test all possible cases of input into our craps game. The user guide and code for our game are accessible at our repo, crapsgame.

iOS Development
ICEE Locator - Created by Emily Lakic & Mark Caldropoli @ BrickHack 3
Won Best Reverse Engineering App

ICEE Locator provides users the ability to find the nearest location carrying ICEE's based on their current location. We built the app using Swift, Python, and Ruby. We used Swift to make the app, Python to scrape the ICEE website in order to obtain a large compiled list of locations totaling 21,145 locations, and Ruby to install Pods for the Google Places API.
